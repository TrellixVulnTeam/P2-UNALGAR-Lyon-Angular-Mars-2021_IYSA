Règles

Début du jeu :

Une première carte avec une date révélée est placée sur la timeline.
Une carte est automatiquement placée dans la main.
Le timer démarre.

Fonctionnement du jeu :

Pour placer une carte il faut appuyer sur le bouton à gauche ou à droite de la carte.
Le jeu valide ou non la position.

Si la position est Incorrecte :
Une pénalité du temps de 2 secondes est réalisée sur le timer.
La carte disparaît du jeu.
Une autre carte est automatiquement placée dans la main.

Si la position est Correcte :
La date de la carte est révélée et reste sur la timeline.
Une autre carte est automatiquement placée dans la main.

Le but est de placer le plus de cartes sur la timeline avant que le timer s'arrête.

Fin du jeu :
La partie s'arrête quand le timer arrive à 0.
Le score est calculé en fonction du nombre de cartes placée sur la timeline.
